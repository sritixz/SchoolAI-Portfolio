Problem Statement 3 Marks
Find the values of x such that the quadratic equation
2x² + 5x - 3 = 0
is satisfied. Show all your working steps clearly.
auto_awesome

Model Solution

1

Factorize the quadratic expression

We need to find two numbers that multiply to (2 * -3) = -6 and add to 5. These numbers are 6 and -1.

2x² + 6x - x - 3 = 0
2x(x + 3) - 1(x + 3) = 0
(2x - 1)(x + 3) = 0
2

Solve for x

Set each factor equal to zero to find the individual roots.

2x - 1 = 0
2x = 1
x = 1/2
x + 3 = 0
x = -3
Final Answer
x = 0.5 or x = -3
