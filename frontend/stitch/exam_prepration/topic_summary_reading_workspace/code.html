Standard Quadratic Form

The standard form of a quadratic equation is ax² + bx + c = 0, where a, b, and c are real numbers and a ≠ 0.

This structure allows us to identify the coefficients needed for the quadratic formula and determines the parabola's direction.

ax² + bx + c = 0

Important Formulas

The Quadratic Formula

x =
-b ± √b² - 4ac 2a

Use this formula when factoring is difficult or impossible. It provides exact solutions for any quadratic equation.

warning

Common Mistakes

cancel The Mistake

Forgetting to include the ± sign before the square root in the formula, leading to only one root instead of two.

x = (-b + √b²-4ac) / 2a
check_circle Correct Approach

Always account for both positive and negative roots. A quadratic equation typically has two solutions.

x = (-b ± √b²-4ac) / 2a
